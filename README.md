#WEBCHAT FOR FAST USER TO USER COMS

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

## DEMO (En proceso)

https://webchatapp-6i08.onrender.com

## Para correr de forma local

Clonar el proyecto

```bash
  git clone https://github.com/lib-paul/WebChat-DEPLOY
```

Ir al directorio del proyecto

```bash
  cd webchatapp
```

Instalar dependencias

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Iniciar el server

```bash
  npm run start
```

Iniciar el cliente

```bash
  //open now terminal
  cd frontend
  npm start
```
